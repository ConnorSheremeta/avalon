# Getting started:

* ```git submodule init```
* ```git submodule update```
* Install Mediainfo cli: http://mediainfo.sourceforge.net
* Copy config/hydrant.yml.example to config/hydrant.yml and change as necessary
* ```rake db:migrate```
* ```rake jetty:start```
* ```rake felix:start```
* ```rake spec```
